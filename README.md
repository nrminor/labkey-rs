# labkey-rs

Unofficial Rust client for the [LabKey Server](https://www.labkey.com/) REST API.

This crate is a port of the official [`@labkey/api`](https://github.com/LabKey/labkey-api-js) JavaScript/TypeScript client, adapted for idiomatic async Rust. It is not affiliated with or endorsed by LabKey Corporation.

## Status

Early development. Not yet published on crates.io.
